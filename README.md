What are some differences between interfaces and types in TypeScript?

TypeScript এ interface এবং type অনেকক্ষেত্রে একইরকম ব্যবহার করা যায় কিন্তু তাদের মধ্যে কিছু গুরুত্বপূর্ণ পার্থক্য আছে ।

What is the use of the keyof keyword in TypeScript? Provide an example.

keyof keyword TypeScript এ ব্যবহার হয় কোনো object type এর সব key এর type বের করার জন্য। সহজভাবে বললে এটা আমাদের বলে এই object এর কি কি property আছে ।
